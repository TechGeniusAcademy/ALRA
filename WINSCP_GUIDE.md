# Инструкция по загрузке на сервер через WinSCP

## Скачать WinSCP
https://winscp.net/eng/download.php

## Настройки подключения

1. Откройте WinSCP
2. Нажмите "Новое подключение"
3. Введите данные:

**Протокол файлов:** SFTP
**Имя хоста:** 89.104.74.76
**Порт:** 22
**Имя пользователя:** root
**Пароль:** yvaAaDiBKCQ8ybN4

4. Нажмите "Сохранить" для сохранения подключения
5. Нажмите "Войти"

## Загрузка файлов

1. После подключения вы увидите два окна:
   - Слева: ваш компьютер
   - Справа: сервер Ubuntu

2. На сервере (справа) перейдите в /root

3. Создайте папку "alra" (правая кнопка → New → Directory)

4. На своем компьютере (слева) откройте папку:
   c:\Users\alkaw\OneDrive\Desktop\Alra

5. Выделите все файлы и папки КРОМЕ:
   - node_modules (в обеих папках client и server)
   - .git
   - logs
   - uploads (если есть старые файлы)

6. Перетащите файлы на правую сторону в папку /root/alra

7. Дождитесь окончания загрузки

## Дальнейшие действия

После загрузки файлов:

1. Откройте PuTTY (или любой SSH клиент)
2. Подключитесь к серверу:
   - Host: 89.104.74.76
   - Port: 22
   - Username: root
   - Password: yvaAaDiBKCQ8ybN4

3. Выполните команды:

```bash
cd /root/alra
chmod +x *.sh
./setup-server.sh
```

4. После завершения setup-server.sh:
   - Настройте MySQL (следуйте инструкциям в DEPLOYMENT.md)
   - Обновите server/.env.production
   - Запустите ./deploy.sh

## Альтернатива: FileZilla

Если предпочитаете FileZilla:

1. Скачайте: https://filezilla-project.org/
2. Откройте FileZilla
3. Файл → Менеджер Сайтов → Новый сайт
4. Введите данные:
   - Протокол: SFTP
   - Хост: 89.104.74.76
   - Порт: 22
   - Тип входа: Нормальный
   - Пользователь: root
   - Пароль: yvaAaDiBKCQ8ybN4
5. Нажмите "Соединиться"
6. Загрузите файлы так же, как в WinSCP
